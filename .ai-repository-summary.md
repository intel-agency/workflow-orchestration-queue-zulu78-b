  - `fetch_queued_tasks()` - Poll for queued items
  - `update_status()` - Update labels and post comments
  - `claim_task()` - Assign-then-verify locking

- `Sentinel` class - Main orchestrator
- `run_forever()` - Polling loop
- `process_task()` - Task execution
- `_heartbeat_loop()` - Background heartbeat

#### `workflow_orchestration_queue.notifier`

- FastAPI app with `/webhooks/github` endpoint
- HMAC signature verification
- Event triage and queue initialization

## Common Tasks

### Add a New Model

1. Create file in `src/workflow_orchestration_queue/models/`
2. Add Pydantic model class
3. Export from `models/__init__.py`
4. Add tests in `tests/test_*.py`

### Add a New Queue Provider

1. Implement `ITaskQueue` interface
2. Add to `src/workflow_orchestration_queue/queue/`
3. Export from `queue/__init__.py`
4. Update `get_queue()` dependency in notifier

### Run Validation Locally

```powershell
# Install tools first
pwsh -NoProfile -File ./scripts/install-dev-tools.ps1

# Run all checks
pwsh -NoProfile -File ./scripts/validate.ps1 -All
```
